HTML(HyperText Markup Language). That's what most internet pages are made from. So is this page. HTML is commonly used with CSS(Cascading Style Sheet) and JS(JavaScript). HTML (along with CSS) is what makes text appear on this page. It is also what makes images, audio, buttons, and more appear. Anyways, lets get started.

Tags

HTML is made up of parts, called elements. An element is represented by tags, which look like this: <html> and </html> The only differences between a start and end tag is the start tags can have attributes, and the end tag has a slash(/) before the tag name. A start tag must also have a matching end tag.

DOCTYPE declaration

When you start typing a HTML file, the version has to be determined. In HTML5, the declaration is <!DOCTYPE html>.
